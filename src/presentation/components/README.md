# UI Components

This folder contains reusable React components for the user interface.

## Purpose
Build modular, reusable UI components that can be composed to create pages and features.

## Examples
- Button
- Input
- Card
- Modal
- Navigation
- Layout components
- Form components

## Guidelines
- Create small, focused components
- Use composition over inheritance
- Implement proper TypeScript types for props
- Follow atomic design principles when applicable
- Keep components pure and testable
- Use CSS modules or Tailwind for styling
- Document complex component APIs
- Organize by feature or type (atoms, molecules, organisms)
