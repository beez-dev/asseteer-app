# Presentation Controllers

This folder contains client-side controllers for managing component state and presentation logic.

## Purpose
Handle client-side logic for complex components or pages, coordinating between UI components and application services.

## Examples
- FormController (handle form submission logic)
- DataTableController (manage table state and filtering)
- DashboardController (coordinate dashboard widgets)

## Guidelines
- Separate presentation logic from UI rendering
- Handle client-side validation and state management
- Coordinate between multiple components
- Call application services or API clients
- Transform data for display
- Use React hooks for state management
