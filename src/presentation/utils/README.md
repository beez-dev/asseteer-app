# Presentation Utils

This folder contains UI utility functions and helpers.

## Purpose
Provide reusable utilities for formatting, validation, and common UI operations.

## Examples
- Date/time formatters
- Number formatters (currency, percentages)
- String utilities (truncate, capitalize)
- Validation helpers
- Class name utilities (clsx, classnames)
- Color utilities
- Browser detection helpers

## Guidelines
- Keep utilities pure functions
- Focus on presentation concerns
- Make utilities easily testable
- Document edge cases
- Use TypeScript for type safety
- Create small, focused utilities
