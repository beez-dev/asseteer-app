# Application Utils

This folder contains application-level utility functions and helpers.

## Purpose
Provide reusable utilities that support application logic, request handling, and data transformation.

## Examples
- DTO mappers and transformers
- Request validators
- Response formatters
- Pagination helpers
- Error handlers

## Guidelines
- Keep utilities pure and reusable
- Focus on application-level concerns
- Avoid domain or infrastructure specifics
- Make utilities easily testable
- Document complex transformation logic
