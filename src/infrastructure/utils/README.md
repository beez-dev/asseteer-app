# Infrastructure Utils

This folder contains infrastructure-level utility functions and helpers.

## Purpose
Provide utilities for infrastructure concerns like configuration loading, connection helpers, and data transformation between external systems and domain models.

## Examples
- Environment configuration loader
- Database connection helpers
- API request/response transformers
- Encryption/decryption utilities
- File system helpers
- Logger configuration

## Guidelines
- Focus on technical infrastructure concerns
- Keep utilities reusable and testable
- Handle environment-specific configurations
- Provide type-safe utilities
- Document external dependencies
