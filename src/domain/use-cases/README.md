# Domain Use Cases

## Purpose
This directory contains pure domain-level use cases that encapsulate core business operations without any application or infrastructure concerns.

## Contents
- Core business operation definitions
- Domain-level business rules
- Pure domain logic use cases
- Business capability interfaces

## Guidelines
- Define pure business operations
- Should be technology-agnostic
- No dependencies on frameworks or external systems
- Focus on what the business can do
- Return domain entities or value objects

## Examples
- CalculateOrderTotal
- ValidateBusinessRule
- ApplyDiscountPolicy
- DetermineEligibility
